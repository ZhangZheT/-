# 定时关机工具 - README.md 说明（直接复制使用）
## 项目介绍
这是一款**Windows系统定时工具**，基于Python开发，可自定义**定时关机、重启、睡眠、锁屏**，操作简单、轻量无广告，适配日常电脑自动管控需求。

## 功能特性
- ✅ 定时关机：自定义时间一键触发电脑关机
- ✅ 定时重启：支持延时重启电脑
- ✅ 睡眠模式：快速进入系统睡眠，节省电量
- ✅ 一键锁屏：即时锁定电脑屏幕，保护隐私
- ✅ 轻量运行：无后台常驻，占用资源极低

## 运行环境
- 系统：Windows 10 / Windows 11
- 语言：Python 3.8+
- 依赖：无第三方库，调用系统原生指令

## 使用方法
1. 下载项目源码
2. 安装Python环境
3. 运行主程序 `main.py`
4. 选择对应功能，设置时间即可执行
5. 可直接打包成exe，直接在项目文件夹打开cmd窗口，输入打包命令即可
